Auth Server

A robust authentication server built with Go, providing JWT-based authentication and user management capabilities similar to Auth0.

Key Features

JWT Authentication

Secure token-based authentication with refresh token support and middleware protection.

User Management

Complete user registration, authentication, and profile management system.

PostgreSQL Integration

Robust data persistence with GORM ORM and PostgreSQL database.

Security First

Password hashing with bcrypt, protected routes, and comprehensive security features.

Installation

# Clone the repository
git clone <repository-url>
cd auth-server

# Install dependencies
make deps

# Configure environment
cp .env.example .env

# Create database
make create-db

# Run the server
make run

API Endpoints

POST /auth/register

Register a new user account.

{
    "email": "user@example.com",
    "password": "password123",
    "first_name": "John",
    "last_name": "Doe"
}

POST /auth/login

Authenticate and receive JWT tokens.

{
    "email": "user@example.com",
    "password": "password123"
}

GET /api/profile

Get authenticated user profile.

Authorization: Bearer <your-access-token>
